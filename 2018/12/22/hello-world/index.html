Membuat Blog Dengan Hexo

2018-12-22 tutorial PV:


Disini saya mau membahas tentang bagaimana sih membuat blog dengan Hexo, untuk penjelasan lebih lanjut tentang Hexo tidak akan saya tulis disini, jadi silahkan baca sendiri ya. Dokumentasi tentang Hexo bisa dibaca disini. Jika ada masalah saat menggunakan Hexo bisa dibaca disini. Langsung aja kita mulai. Oiya, pastikan di sistem kita sudah terinstall NodeJS dan npm, cara installnya bisa dibaca disini untuk NodeJS dan disini untuk npm.

Tutorialnya

Install Hexo dan Inisialisasi Blog

Kita install Hexo di sistem kita dengan perintah ini.

lalu buat folder yang akan kita jadikan repository lokal blog kita.

namablog itu bisa kalian ganti sesuai keinginan, selanjutnya masuk ke direktori blog kalian.

selamat kalian sudah berhasil membuat blog kalian dengan Hexo, lanjut ke tahap selanjutnya.

Menambah Tema

Kita pasti ingin dong tampilan blog kita bagus, oleh karena itu kita perlu menambahkan tema pada blog kita, Hexo sendiri sudah menyediakan tema untuk blog kita, jadi kita tidak perlu membuat tampilan sendiri dengan HTML dan CSS, semua tema itu bisa kita akses disini. Lalu clone tema dengan perintah.

selain dengan perintah clone bisa juga diunduh dan di ekstrak ke dalam direktori themes, jangan lupa membaca dokumentasi dari setiap tema karena pengaturan tiap tema berbeda-beda.

Membuat Postingan

Untuk membuat postingan kalian bisa menggunakan perintah ini.

secara otomatis, Hexo akan membuat file “judul postingan.md” pada direktori source/_post. Lalu jika kita ingin membuat sebuah draft atau halaman khusus untuk blog kita, kita bisa menggunakan perintah.

atau

penjelasan lainnya tentang membuat postingan bisa kalian baca disini.

Menjalankan Server

Dengan Hexo, kita dapat melihat preview blog kita sebelum diluncurkan atau kita deploy ke GitHub dengan menggunakan perintah ini.
$ hexo server atau $ hexo server --draft untuk melihat preview blog kita beserta file yang masih menjadi draft. Perintah ini akan membuka tautan http://localhost:4000/ dan kita dapat membukanya melalui browser kesayangan.

Generate dan Deploy

Langkah terakhir yaitu generate dan deploy, perintah ini akan mempublish tulisan kita ke repo GitHub agar dapat ditampilkan seperti blog pada umumnya, tapi sebelum itu kita install dahulu plugin hexo-deployer-plugin agar mempermudah proses deploy, caranya cukup jalankan perintah ini $ npm install hexo-deployer-git --save . Jika sudah, kita edit berkas _config.yml dan tambahkan data ini di dalamnya.

lalu kalian bisa menjalankan perintah generate dan deploy

Informasi lainnya bisa kalian baca disini Deployment dan disini Generate.
